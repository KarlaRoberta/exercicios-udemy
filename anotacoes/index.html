Anotações do curso de JavaScript

  • Parâmetros e retornos são opcionais
  • this - acessa o 'dono' do código executado naquele momento.
    OBS.: Quando você tem uma função tradicional, em JavaScript, usando a palavra "function", o "this" pode variar de acordo com quem chamou a função. Quando usamos o "arrow", o "this" não varia.
  • Função arrow é mais reduzida "this" é associado ao local onde a função foi escrita, associado com contexto lexico.
  • funções anônimas - função sem nome
  • função callback
  • função construtora
  • contexto lexico - como as coisas foram declaradas
  • Closure é o escopo criado quando uma função é declarada. Esse escopo permite a função acessar e manipular variáveis externas à função
  • função factory - função que retorna um objeto
